# Task Management App

## Project Description

This project is a simple Task Management App created using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, delete them, and keep their tasks saved using Local Storage. The project was also used to practice Git and GitHub collaboration by working with branches, commits, and pull requests.

## Implemented Features

- Display initial tasks
- Add a new task
- Prevent adding empty tasks
- Mark tasks as completed or incomplete
- Delete tasks with a confirmation message
- Show the number of incomplete tasks
- Save tasks in Local Storage
- Keep tasks after refreshing the page

## Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub

## How to Run the Project

1. Clone the repository from GitHub.
2. Open the project folder.
3. Open the `index.html` file in any web browser.
4. Start using the application.
